This is an app programmed for QT. In order to run this app, you need Qt Creator 4.4.1 or upwards. 

You will need to download all the files in a folder, and open the folder with QT Creator. In order to run the app, you will also need to install a c++ compiler.

The figures that are drawn with this app are:
-Polygons
-Arcs
-Cubes
-Prisms
-Cones

The app also permits a series of rotations, zooming in/out, translations, and reflections. All the algorithms programmed for this are clearly structured.
